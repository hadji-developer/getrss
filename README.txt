Данное консольное php-приложение парсит rss-ленту новостей 'http://static.feed.rbc.ru/rbc/logical/footer/news.rss'
и вносит информацию об этих новостях в базу данных.

Приложение тестировалось на:
    операционная система - linux debian 10.9;
    база данных - mariadb 10.3.27;
    php - версия 7.3.

Так как приложение консольное, в системе должен быть установлен пакет php-cli. Сам php интерпретатор должен быть доступен в терминале при вводе команды "php".
Eсли нет доступа, добавьте в системную переменную $PATH папку, где находится ваш исполняемый файл php (в debian 10 это обычно /usr/bin/php).
Для работы приложения требуется пакет php-curl.
Предварительно следует создать базу данных. БД тестировалась на mariadb 10.3.27 и возможно будет работать без ошибок на mysql 8.
Дамп базы данных находится в папке mysql_dump.

----------ЗАПУСК----------------------------
Пройдите в терминале в папку, куда вы это приложение клонировали или загрузили. Далее наберите в терминале

    $ cd entry_point

Команда запуска

    $ php rbcrss.php

При успешном выполнении приложение сообщит о времени исполнения запроса и его обработки.
Если сообщения нет - значит запрос не удался. Отчет о запросах можно получить, набрав команду

    $ php rbcrss.php --requests 10
    
Будет выведена информация о 10 последних запросах. Можно указать любое количество запросов.
Если не указать - то будет выведена информация о последнем запросе.

Вывести  в терминал последние новости можно командой

    $ php rbcrss.php --last 5

Соответственно приложение выведет 5 (или то количество, которое вы укажите) последних новостей, если таковые есть.
Если вы не укажите количество последних новостей, то будет выведена лишь одна последняя новость.


Для периодического запуска через cron введите команду
    $ crontab -e
И в редакторе введите следующую строку
    */30 * * * * php полный путь до папки, куда вы установили приложение/entry_point/rbcrss.php
После сохранения каждые полчаса база данных будет пополняться новостями от rbc.
